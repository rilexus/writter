# writer


## Description

Application to manage personal notes.

## Dependencys
* Angular
* NestJS
* MongoDB(Mongoose)

## Installation

```bash
$ npm install
$ cd ./writer
$ npm install
```

## Running the app

```bash
# development
$ npm run start:client
$ npm run start:webpack
$ npm run start:db
$ npm run start:server
```
