# Writer


## Description

Application to manage personal notes.

## Dependencys
* Angular
* NestJS
* MongoDB(Mongoose)


## Installation
```bash
$ npm install
$ cd ./writer
$ npm install
```

## Running development env
```bash
# start the client
$ npm run start:client

# start the database
$ npm run start:db

# start the server
$ npm run start:server

# start the nest hot reload
$ npm run start:webpack
```
